## file explore
type vim (no file)
: Explore
: Lex  # best option
: Sex
: Hex

splits the screen (left - file explorer, right - vim editor panel)
use up / down arrow buttons to navigate through files and directories on the left
use enter button on file so it enters on right pane

use Ctrl W (press W twice to toggle between left and right pane)
when you enter a different file, it replaces the open file on the right pane
